Module: linearArrayFlattener

Linear function to flatten a multi-dimensional array of arbitrary depth. Linear time of O(n) without recursion. Supports IE > 5. Based on axelduch's Stack Overflow answer https://stackoverflow.com/a/27282907. axelduch provided permission for modified or unmodified redistribution.
Parameters:
Name Type Attributes Default Description
array Array The array to be flattened.
mutable boolean <optional>
 false Specify mutability of array. true: Memory complexity of O(1). false: Memory complexity O(n).
